String Comparisons

There are three ways to compare a String in Java

  1. equals() method
  2. compareTo() method
  3. == operator: Compares references not values. Caution: Only works if using same string constant pool instance.
    4. String s1="Welcome"; // Creates new instance
    String s2="Welcome"; // Doesn't create new instance
