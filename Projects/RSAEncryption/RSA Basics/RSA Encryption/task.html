RSA Encryption

RSA encryption allows coded messages to be sent without the sender (or any third-party) knowing how to decode the message. This encryption works using a public keys that is known to everyone and a private key that is known only by the receiver.

Public Key <n, e>: Encode Message
  1. Select two distinct primes, p and q. We'll use primes from 127 to 200.
  2. Multiply p and q to get n, the encryption modulus.
      n = p * q
  3. Select prime e that is less than n and relatively prime to (p-1) * (q-1).
      φ(n) = (p-1) * (q-1)
      1 < e < φ (n)
      gcd(e, d (n)) = 1   i.e. φ (n) % e != 0
  4. Convert message String to ASCII values bytes from plain text P.
      byte[] M = P.getBytes(StandardCharsets.UTF_8);
  5. Encrypt plain text m byte into cipher text C using public key <n, e>.
      For each byte in M: m = byte
      C ≡ me mod n
Private Key<n, d>: Decode Message
  1. Determine private key d.
      (k*d) mod φ(n) ≡ 1
      d = (k * φ(n) + 1)/e   [let k = 0, 1, 2, 3 ...]
        i.e. Find lowest k where (k * φ(n) + 1) mod e ≡ 0
  2. Decrypt cipher text C from into decoded byte text m using private key <n, d>.
      For each byte in C: c = C
      M ≡ cd mod n
  3. Convert decoded byte text from ASCII values M to plain text P.
      String P = new String(M, StandardCharsets.UTF_8);
