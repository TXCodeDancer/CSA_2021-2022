Cat Class: Instance Variables & Methods

Implement the Cat class by adding instance variables (attributes) and methods (behaviors).

Implement the following instance variables (attributes):

  • name as a string.
  • age as an int.
  • color as a string.
  • isSleeping as a boolean

Implement the following methods (behaviors):

  • Public Cat constructor without parameters:
    • Set name = ""
    • Set age = Integer.MIN_VALUE
    • Set color = ""
    • Set isSleeping = false
  • Public Cat constructor with parameters: public Cat(String name, int age, String color)
    • Set name = name
    • Set age = age
    • Set color = color
    • Set isSleeping = false
  • Public sleeps() method:
    • Set isSleeping = true
When a class method parameter has the same as the class instance variable, use this.varName = varName to set the class instance variable. For example:
    this.name = name;
