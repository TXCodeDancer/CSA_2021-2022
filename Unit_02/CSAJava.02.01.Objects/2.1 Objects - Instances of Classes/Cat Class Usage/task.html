Cat Class: What are Objects and Classes?

The goal of this task is to help you become familiar with Java classes and using instances of Java classes called objects. this task includes two classes, Main and Cat. Your objective is to write the missing code in the Main class to:

  • Output information for each cat instance to the console.
  • Modify the status of the cat named Ben to record that he is sleeping.
Review how the default cat's information was output using System.out.println().
Review how the panda cat status was set to sleeping.
